# Airent - Sistem Informasi Rental Kebaya

Project ini dibuat untuk memenuhi tugas UAS Matakuliah Pemrograman Web.
Website ini dibangun menggunakan PHP Native, MySQL, dan Bootstrap 5.

## Fitur Website
1. Halaman Frontend (Katalog & Cara Sewa).
2. Fitur Zoom Gambar (Lightbox).
3. Form Penyewaan Online.
4. Halaman Admin (Login & Logout).
5. CRUD Data Kebaya (Create, Read, Update, Delete).
6. Manajemen Data Pesanan Masuk.

## Cara Instalasi
1. Download repository ini.
2. Extrak ke folder `htdocs`.
3. Buka PHPMyAdmin, buat database baru bernama `db_airent`.
4. Import file `db_airent.sql` yang ada di folder project.
5. Buka browser: `localhost/airent`.
